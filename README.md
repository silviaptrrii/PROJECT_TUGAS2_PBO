Mata Kuliah: Pemrograman Berorientasi Objek

Deskripsi:
Program ini adalah simulasi sederhana manajemen penumpang bus menggunakan bahasa Java. Program dibuat untuk memenuhi tugas projek individu dengan menerapkan konsep OOP.

Fitur Utama:
1. Mengatur penumpang naik dan turun.
2. Logika kursi prioritas untuk Lansia, Anak-anak, dan Ibu Hamil.
3. Sistem pembayaran ongkos dan pengecekan saldo.
4. Validasi kapasitas maksimal bus (40 penumpang).

Konsep OOP yang digunakan:
Class, Object, Encapsulation, Constructor, ArrayList, dan Method.
